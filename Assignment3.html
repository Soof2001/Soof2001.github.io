Week 1.4 - Housing

The city of Amsterdam expects a lot of visitors for the open water swimming event. To have an overview of the accommodation possibilities for the extra tourists entering the city, Amsterdam wants to get an insight into the number of tourists that will make use of Airbnb.

On this webpage we will provide information on the research background, methodology and used datasets for Assignment 3. In the Jupyter source file ‘Assignment3_WB, which can be found in the Assignment 3 Zip File on GitHub, we have structured our code and Python script. These Python codes can be run and reproduced in the Jupyter notebook.

Tourist tax

To calculate the amount of income that Amsterdam expects to generate from tourist tax when 30.000 visitors will come to the event for the whole week, we have used data on tourist tax from the municipality of Amsterdam (2023). According to this information, an Airbnb location can ask 10% tourist tax of the price per night. To know what the average price of an Airbnb location is per night, we used data from ‘InsideAirbnb’, which is an Airbnb scraper website. We have downloaded the ‘listings.csv’ file, which summarizes information and metrics for Airbnb listings in Amsterdam in 2023. From this dataset, we calculated the mean price of an Airbnb per night and multiplied it by the tourist tax per night. The total income for Amsterdam for the whole week was generated by multiplying this number by 30.000 visitors and 7 days. Our exact coding can be found in Assignment3_WB. In total, the Amsterdam municipality could generate about 5.350 000 euros on tourist tax.

Cross-reference data Airbnb dataset and BBGA data

We created a cross reference between the data from the InsideAirbnb (2023) dataset and the Basis Bestand Gebieden Amsterdam (BBGA) dataset from Gemeente Amsterdam (2023). The BBGA dataset contains key figures at the level of the most commonly used area divisions in Amsterdam such as neighbourhoods, shopping and working areas. We have downloaded the data set as a XLSX file. Our trajectory to creating the map with plotted Airbnb locations per neighbourhood can be found in our workbook Assignment3_WB. The result of this plot is the following:

The BBGA dataset is in dark blue, whereas the Airbnb dataset is in red. Unfortunately, we did not manage to create a proper legend next to the plot.

Presumably, the higher results in the BBGA dataset can be attributed to its continuous data accumulation over the course of the year, resulting in an annual cumulative representation. In contrast, the Airbnb dataset constitutes a temporal snapshot, reflective of the platform’s status at the time of the data retrieval. Consequently, it may omit listings present in the BBGA dataset due to their removal from the Airbnb platform subsequent to the data collection point.

Airbnb apartments used for housing

To determine whether or not an Airbnb is also used as a normal house, we have logically assumed that if an Airbnb is used for more than 200 days as an Airbnb location, it is not used as a normal house. Our code can be found in the Assignment3_WB. From our Jupyter notebook script we can conclude that 6760 Airbnb’s are not rented out all the time but are also used as normal housing, and 1626 are not used as housing.
