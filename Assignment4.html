Based upon the information from assignment 1 about the general surface waters, commercial water transport ways, canal boat tours and pollution levels, we created a map in Python Jupyter notebook which provides an overview of the canals which are most suitable for hosting the event. To create this map, we have used json geodata from MapBox (2023).

It can be concluded that this 5km route is possible as it covers the potential waterways that are not used for commercial water transport and as routes for the canal boats.

Central location and bus/tram transport stops and lines

The municipality of Amsterdam wants to have a good location for the Event Headquarters and they want to regulate the afterparty and the stream of visitors. In our Jupyter source file ‘Assignment4_WB’, which can be found in the Assignment 4 Zip File on GitHub, we have structured our code and Python script. These Python codes can be run and reproduced in the Jupyter notebook. In this Assignment4_WB we have plotted the center of the route; the closest bus and tram stops at the start and finish of the swimming route; how many people can be transported within an hour and the bus and tram lines. To plot the closest bus and tram stops, we have used the geodataset ‘TramMetro_Punten_2022’ from the municipality and downloaded it as a json file. To discover where the bus and tram lines are surrounding the event, we looked at the geodataset ‘TramMetro_Lijnen_2022’, which can also be downloaded as a json file from the Amsterdam municipality website.
